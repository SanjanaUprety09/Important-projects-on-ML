**Project Title:** Disease Prediction using Machine Learning
**Description:**
In this project, I built a machine learning model to predict diseases based on symptoms. I started by preprocessing the dataset using label encoding and handled class imbalance using RandomOverSampler. I trained three different models: Support Vector Machine (SVM), Gaussian Naïve Bayes, and Random Forest. To evaluate the models properly, I used stratified K-Fold cross-validation.

**🔹 Model Performance Highlights:**

Random Forest gave the best results with around 95–98% accuracy.

SVM also performed well with 90–94% accuracy.

Naïve Bayes had lower accuracy but was still decent, around 85–90%.

Using oversampling really helped improve the prediction accuracy, especially for less common diseases.

I compared the models using accuracy, precision, recall, and F1-score.

The results were consistent across folds, showing that the models are quite reliable.

**Overall**, this project helped me understand how to deal with imbalanced datasets, evaluate classification models, and improve performance using basic ML techniques.
